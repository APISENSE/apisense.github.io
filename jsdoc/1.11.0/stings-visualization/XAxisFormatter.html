XAxisFormatter

new XAxisFormatter()

Formatter for the x axis

Available formatters to set for labels in ChartWidget:
  • DATE_FORMAT
    • Date formatter, x values are timestamps
  • STRING_FORMAT
    • String formatter, x values are strings
  • DEFAULT_FORMAT
    • Default formatter, x values are float numbers
